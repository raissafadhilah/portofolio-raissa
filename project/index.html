Tentang Saya

Halo! Saya Raisa Fadhilah, mahasiswa Informatika di Horizon University yang aktif dan antusias dalam dunia perangkat lunak dan teknologi digital. Sejak awal perkuliahan, saya telah mendalami berbagai aspek pengembangan perangkat lunak, mulai dari rekayasa perangkat lunak, pengujian aplikasi, hingga pengembangan web berbasis Laravel dan JavaScript.

Saya percaya bahwa perangkat lunak bukan hanya soal baris kode, tapi tentang menciptakan solusi nyata yang berdampak positif bagi kehidupan. Oleh karena itu, saya terus mengasah keterampilan melalui berbagai proyek mandiri dan kolaboratif, seperti pembuatan aplikasi perpustakaan digital, media pembelajaran interaktif, hingga website portofolio pribadi ini.

Selain kemampuan teknis, saya juga terbiasa bekerja secara terstruktur, cepat belajar, dan terbuka terhadap teknologi baru. Portofolio ini adalah cerminan dari perjalanan dan semangat saya dalam dunia teknologi.
